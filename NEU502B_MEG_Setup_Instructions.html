00 Prerequisites
Mac & Linux — Install uv
🍎 Mac & Linux
Terminal 
curl -LsSf https://astral.sh/uv/install.sh | sh
💡
Windows users are strongly encouraged to install WSL (Windows Subsystem for Linux), which allows your machine to run Linux. From PowerShell (run as Administrator):
PowerShell — run as Administrator 
wsl --install

This installs WSL 2 with Ubuntu by default. A reboot is required afterward. Once WSL is set up, follow the Mac & Linux instructions above instead of the Windows steps below.

ℹ️ The steps below only apply to Windows users not running WSL. Mac and Linux users can skip to Part 1.
1. Install uv

From PowerShell (you may see a security prompt — this is expected and safe to proceed through):

PowerShell 
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
2. Install Git

From Windows Terminal:

Windows Terminal 
winget install Git.Git
3. Install Anaconda or Miniconda

Download from anaconda.com if you don't already have it, then open Anaconda Prompt and run:

Anaconda Prompt 
conda install pyqt
conda install jupyterlab
Part 01 Get the mne-opm Software
1. Clone the repository

Navigate to your preferred software location first, then clone:

Terminal 
git clone -b my-working-branch https://github.com/mrribbits/mne-opm.git
2. Enter the directory and sync dependencies
Terminal 
cd mne-opm
uv sync
3. Register the Jupyter kernel
🪟 Windows (not WSL)
Anaconda Prompt — from mne-opm directory 
.venv\Scripts\activate
pip install ipykernel
python -m ipykernel install --user --name mne-opm --display-name "mne-opm"
🍎 Mac or Linux (if you have kernel issues)
Terminal — from mne-opm directory 
uv run python -m ipykernel install --user --name mne-opm --display-name "mne-opm"
Part 02 Download the Preprocessed Sample Data
1. Navigate to your preferred data location and download
Terminal 
curl -L -H "Cache-Control: no-cache" \
  -o "./MEG-for-students-preproc.zip" \
  "https://www.dropbox.com/scl/fi/mwk2xs6jcar5ppl6ahk0t/MEG-for-students-preproc.zip?rlkey=0bpthyvizakg1h8c9ncy10sp4&dl=1"
2. Unzip the file
Terminal 
unzip MEG-for-students-preproc.zip
Part 03 Set Up the Jupyter Notebooks
1. Clone the course notebooks
Terminal 
git clone https://github.com/mrribbits/NEU502-2026.git meg-notebooks
2. Launch JupyterLab

Point uv at your mne-opm install location:

Terminal — edit the path to match your mne-opm location 
uv run --project /path/to/your/mne-opm jupyter lab
💡 Once open, make sure the mne-opm kernel is selected. If it isn't chosen automatically, select it from the kernel menu in JupyterLab.
3. Edit the path variables in each notebook

Open each notebook and update the directory variables to match your local setup:

Variables to edit
DERIV_DIR
Preprocessed derivatives folder — set in Notebooks 1–4 e.g. ~/classes/neu502/oddball/data/oddball/bids/derivatives/analysis1__/sub-001/ses-01/meg
RAW_DIR
Raw BIDS data folder — set in Notebook 1 e.g. ~/classes/neu502/oddball/data/oddball/bids/sub-001/ses-01/meg
SUBJECTS_DIR
FreeSurfer subjects directory — set in Notebook 3 e.g. ~/classes/neu502/oddball/data/oddball/bids/derivatives/freesurfer
PROJECT_ROOT
Top-level project folder — set in Notebook 5 e.g. ~/classes/neu502/oddball
